Window Hider

Ein leistungsstarkes Tool zum Verstecken von Fenstern im Stream. Verstecke einzelne Anwendungen, nicht den ganzen Desktop.

A powerful tool for hiding windows from your stream.

Windows 10/11 Python 3.x Tkinter GUI Open Source

Features

👁

Einzelne Fenster verstecken

Verstecke nur bestimmte Programme, nicht den ganzen Desktop. Ideal für sensible Informationen während des Streamings.

🔄

Auto-Hide

Automatisches Verstecken von bestimmten Programmen beim Start. Nie wieder vergessen, private Fenster auszublenden.

📥

System Tray

Läuft diskret im Hintergrund im System Tray. Schneller Zugriff über das Tray-Icon.

🎨

Moderne UI

Schickes schwarzes Design mit flüssiger Bedienung. Integriertes Settings Panel für alle Einstellungen.

⚙

Settings Panel

Integrierte Einstellungen direkt in der App. Konfiguriere Auto-Hide, Opazität, Tray-Verhalten und mehr.

📱

Kompaktes Design

Platzsparende, vertikale Listenansicht mit Scroll-Funktion. Übersichtlich auch bei vielen Fenstern.

Schnellstart

1

Admin starten

Starte die Anwendung als Administrator für vollen Zugriff

2

Fenster wählen

Wähle ein Fenster aus der Liste aller sichtbaren Fenster

3

HIDE klicken

Klicke auf HIDE um das Fenster vor dem Stream zu verstecken

4

SHOW klicken

Klicke auf SHOW um es wieder anzuzeigen
